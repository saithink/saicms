<h1>SaiCms</h1>

基于<a href="https://saithink.top/" target="_blank">saiadmin</a>的内容管理系统

<h1>安装</h1>

composer环境的安装命令如下

``` bash
composer require saithink/saicms
```

安装之前确保已经安装了saiadmin


安装后页面访问路径: http://localhost:8787/app/saicms
